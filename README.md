## Require
Python

Twitter Developer Account

Requests (Python module)

Tweepy (Python module)

## COMMENCEMENT
Télécharger et extraire le bot.
Ouvrir le fichier config.py et faire suivre les instructions. 
Il faut avoir python d'installer et à jour!
Ensuite:

lancer **install.bat**!

Si ça ne marche pas:
Ouvrez CMD et lancer les commandes suivanets!

`pip install requests` 

Alternatively: `pip3 install requests`

`pip install tweepy`

Alternatively: `pip3 install tweepy`

**Maintenant vous pouvez lancer: run.bat!!**
